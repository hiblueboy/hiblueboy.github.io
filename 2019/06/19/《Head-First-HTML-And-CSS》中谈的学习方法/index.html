读书笔记

《Head First HTML And CSS》中谈的学习方法

最近在学前端相关的技术，想从最基础的开始学；在豆瓣找到一本9.3分讲html和css的书—《Head First HTML And CSS》，这么高的评分想必也是入门html和css的红宝书了；书中罗列了9个学习这本书的最佳方法，并强调是非常科学和遵循大脑规律的学习方法：

  1. 慢一点，你理解的越多，需要记的就越少

    2. 不要光看看就行了，停下来好好想一想；书中提出问题的时候，你不要直接去翻答案。可以假想真的有人问你这个 问题。你让大脑想得越深入，就越有可能学会并记住它。

  2. 做练习，自己做笔记

    3. 不做练习直接看答案，那和有人替你考试有什么区别。不要只是坐在那里看着发呆。拿出笔来，写一写画一画。大量研究都证实，学习过程中如果能实际动动手，将改善你的学习。

  3. 阅读"没有傻问题"部分

    4. 这些问题不是可有可无的旁注，他们绝对是核心内容的一部分！千万不要跳过去不看。

  4. 上床睡觉之前不要再看别的书，至少不要看其他有难度的东西

    5. 学习中有一部分是你在合上书后完成的(特别是要把学到的知识长久地记住, 这往往无法在看书的过程中做到)。你的大脑也需要有自己的时间，这样才能做一些处理。如果在这段时间内你又往大脑里灌输了新的知识, 那么你刚才学习的东西就会丢掉。

  5. 要喝水，而且要多喝水

    6. 能提供充足的液体，你的大脑才能有最佳表现。如果缺水(可能在你感觉到口渴之前就已经缺水了)学习能力就会下降。

  6. 讲出来，而且要大声说出来

    7. 说话可以刺激大脑的另一部分。如果你想看懂什么，或者想更牢地记住它，就要大声说出来。更好的办法是，大声地解释给别人听。这样你会学得更快，而且可能会有一些新的认识，而这是以前光看不说的时候未曾发现的。

  7. 听听你的大脑怎么说

    8. 注意一下你的大脑是不是负荷太重了，如果发现自己开始浮光掠影地翻看，或者刚看的东西就忘记了，这说明你该休息一会儿了。达到某个临界点时，如果还一味地向大脑时塞，这对加快学习速度根本没有帮助，甚至还可能影响正常的学习。

  8. 要有点感觉

    9. 你的大脑需要知道这是很重要的东西。要真正融入到书中的故事里。为书里照片加上你自己的说明。你可能觉得一个笑话很蹩脚，不太让人满意，但这总比根本无动于衷要好。

  9. 真正做些工作

    10. 把这应用到你最近设计的网页或者旧项目上。干些事情，以获取书上练习和其他项目以外的经验。你所要做的只是准备一支笔和一个等待解决的问题...嗯，一个对你使用HTML和CSS有帮助的问题。
