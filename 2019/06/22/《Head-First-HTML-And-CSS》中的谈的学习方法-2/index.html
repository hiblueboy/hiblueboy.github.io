读书笔记

《Head First HTML And CSS》中谈的学习方法(2)

这本书一开始讲的内容是关于学习方法和大脑学习的特点，告诉我们大脑渴望是一些新奇的东西，那才是它认为重要的事情，所以能很好地记住，这是大脑对事物最基本的反应。

大脑学习事物的特点

大脑总是渴求一些新奇的东西。它一直在搜寻、审视、期待着不寻常的事情发生。大脑的构造就是如此，正是这一点才让我们不至于墨守成规，能够与时俱进。 大脑真正工作的时候，是记住那些确实重要的事情。它不会费心地去记乏味的东西，就好像大脑里有一个筛子，这个筛子会筛选掉"显然不重要"的东西，如果遇到的事情枯燥乏味，这些东西就无法通过这个筛子。

让大脑认为学习的新东西很重要

如果你想把这本书读到的知识真正用起来，就需要记住你读到的所有内容。为此，必须理解这些内容。要想最大程度地利用这本书或其他任何一本书，或者掌握学习经验，就要让你的大脑负起责来，要求它记住这些内容。技巧就在于要让你的大脑认为你学习的新东西确实很重要，对你的生活有很大影响。就像老虎出现在面前一样。

如何让大脑活动起来

怎么才能让大脑认为需要学习的东西很重要呢？这有两条路，一条比较慢，很乏味。另一条路不仅快，还更有效。慢方法就是大量地重复，如果做了足够的重复，你的大脑就会认为 "尽管看上去好像不重要，但经常地看到同一个东西，就会假定这应该是重要的。" 而更快的方法是尽一切可能让大脑活动起来，特别是开动大脑来完成不同类型的活动。例如：

  • 把文字放在所描述图片的中间，这样会让你的大脑更多地考虑这些文字与图片之间有什么关系，而这就会让更多的神经元"点火"—你的大脑更有可能认为这些内容值得关注，而且很可能需要记下来。
  • 交谈式风格也很有帮助，当人们意识到自己在与"别人"交谈时，往往会更加专心，这是因为他们总想跟上谈话的思路，并能做出适当的发言。让人惊奇的是，大脑并不关心"交谈"的对象究竟是谁，即使你只是与一本书"交谈"，它也不会在乎！说话的方式最好是讲故事似的，要用通俗的语言，不能太严肃，不要像在做报告。
